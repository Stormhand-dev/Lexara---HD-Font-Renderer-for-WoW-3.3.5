Lexara is a lightweight DLL that replaces WoW 3.3.5's native font 
renderer with MSDF (Multi-channel Signed Distance Field) technology, 
delivering sharp, resolution-independent text across the entire game UI — 
combat numbers, nameplates, tooltips, chat, and more.

Drop dinput8.dll into your WoW root folder. No addon, no configuration 
file, no setup required.

Built on the font subsystem of Transmorpher 2.0.0 by AwesomeWotLK, 
stripped to a single focused purpose.
